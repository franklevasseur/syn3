## Usage

### Requirements

- Node.js v16.15.1 (you can use [nvm](https://github.com/nvm-sh/nvm))
- Yarn (npm i -g yarn)

### Installation

```bash
git clone git@github.com:franklevasseur/syn3.git
cd syn3
yarn
yarn build
yarn start
```
